# 🤖 Daily Lead Generation System

Automatically scrapes 15+ small business leads daily, checks for duplicates, and adds them to Google Sheets for your cold outreach pipeline.

## 📊 Workflow

```
┌─────────────────────────────────────────────────────────────┐
│  GitHub Actions (Runs Daily at 7 AM)                        │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Lead Scraper Python Script                                  │
│  • Yell.com directory (primary)                             │
│  • Companies House API (secondary)                          │
│  • DuckDuckGo search (email finding)                        │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Duplicate Check                                             │
│  • Fetch existing emails from Google Sheet                  │
│  • Filter out duplicates                                     │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Google Sheet Update                                         │
│  • Append 15 new leads                                       │
│  • Format: Name | Business | Email | Date | Status...       │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  Make.com Watches Sheet (Your Existing Setup)               │
│  • Detects new rows                                          │
│  • Triggers 3-email outreach sequence                        │
└─────────────────────────────────────────────────────────────┘
```

## ⚡ Quick Start

1. **Read SETUP_GUIDE.md** (5-minute setup)
2. Set up Google Service Account
3. Add GitHub Secrets
4. Upload files to your repo
5. Test with "Run workflow"
6. Done! Leads arrive daily at 7 AM
